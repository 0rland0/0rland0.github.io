Web Messaging

Using a Web SDK developed by Smooch, this integration lets your customers message you from your web page, while you reply using your favorite business tool.

It also connects with other messaging channels to engage with customers wherever they are.

For additional help or to install the Web SDK using a different method, follow the full installation guide.

STEP 1

Choose a Method

Choose your preferred method to include the Smooch plugin on your web page.

STEP 2

Include the Smooch plugin on your web page

Add the following code towards the end of the <body> section on your page

<script src="https://cdn.smooch.io/smooch.min.js"></script>
STEP 3

Initialize Smooch with your App Token

Once Smooch has been added to your webpage, you're almost done. Simply initialize the plugin using this code snippet. This should be placed after the code in Step 2.

<script>
Smooch.init({ appToken: '0zl8bf0s24oomxn13tu663rpw' });
</script>
STEP 4

Confirm your Integration

We'll run a quick check to verify that your integration is up-and-running.
